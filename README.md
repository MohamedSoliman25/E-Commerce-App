# E-Commerce-App

E-commerce demo application that allows users to purchase products online and is provided with the electronic Razor Payment system.
####
## Tools

- Firestore Database
- MVVM
- Live Data
- Razor Payment Api
- Image Slider
- Material Design
- Glide
